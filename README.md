Numbers — простое Android-приложение «Угадай число». При запуске загадывается случайное число от 1 до 5. Пользователь вводит число и получает подсказку: угадал он, либо загаданное число больше или меньше. Приложение предназначено для учебной практики и знакомства с основами Android-разработки на Java.
